# Blog Draft Uploads

Use this folder to drop raw blog drafts before moving them into production learning content.

## How to use

1. Duplicate `_scheme-blog-template.md`.
2. Rename the copy using the target scheme and state, for example:
   - `nsw-fhbas-draft.md`
   - `federal-fhss-draft.md`
3. Paste your generated draft into the file.
4. Keep placeholders and caveats if facts are still being verified.
5. After review, move final content into `content/learn/schemes/*.mdx`.

## Notes

- This folder is intentionally not rendered in the web app.
- Keep these files as working drafts.
